🔌 API ENDPOINTS
════════════════════════════════════════════════════════════════════════════

Health Check
  GET http://localhost:5000/health
  Returns: {status: "healthy", model_loaded: true}

Single Prediction
  POST http://localhost:5000/predict
  Body: {spo2: 95, respiratory_rate: 18, ...11 fields}
  Returns: {risk_level: "LOW", confidence: 0.9974, ...}

Batch Prediction
  POST http://localhost:5000/batch
  Body: [{patient1}, {patient2}, ...]
  Returns: {total: 2, predictions: [...]}

Example Patients
  GET http://localhost:5000/example-patients
  Returns: {healthy: {...}, high_risk: {...}}

Model Info
  GET http://localhost:5000/model-info
  Returns: {model_type: "RandomForest", features: [...], ...}

🔧 TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════

Problem: "Server Status: ❌ Disconnected"
Solution: Make sure Flask is running in Terminal 1
          Check: curl http://localhost:5000/health

Problem: "Port 3000 already in use"
Solution: Kill process: lsof -ti:3000 | xargs kill -9
          Then: npm start

Problem: "Port 5000 already in use"
Solution: Kill process: lsof -ti:5000 | xargs kill -9
          Then: python api_server.py

Problem: "Module not found" errors
Solution: Reinstall: npm install
          Or: npm install axios recharts

Problem: CORS error
Solution: Restart both servers (Ctrl+C, then start again)
          Flask has CORS enabled by default
